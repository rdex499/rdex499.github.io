Objective

The goal of this lab is to familiarize ourselves with the Artemis board and set up the development environment. We'll test basic functionality including the onboard LED, serial communication, and sensor readings.

Note: Make sure you have Arduino IDE 2.0+ installed before starting.

Setup

First, we need to install the SparkFun Apollo3 board support in Arduino IDE:

  1. Open Arduino IDE
  2. Go to File → Preferences
  3. Add the board manager URL
  4. Install SparkFun Apollo3 boards

Task 1: Blink It Up

The classic "Hello World" of embedded systems. Let's make the onboard LED blink!

Code

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);
  delay(1000);
  digitalWrite(LED_BUILTIN, LOW);
  delay(1000);
}

Task 2: Serial Communication

Testing serial communication at different baud rates. Using Serial.begin(115200) for optimal performance. 

void setup() {
  Serial.begin(115200);
  Serial.println("Artemis board initialized!");
}

void loop() {
  if (Serial.available()) {
    char c = Serial.read();
    Serial.print("Received: ");
    Serial.println(c);
  }
}

Challenges

Challenge: Initial upload failed due to incorrect board selection.

Solution: Selected "SparkFun RedBoard Artemis Nano" instead of the generic Apollo3 board. This resolved the upload issues.

Conclusion

Successfully set up the Artemis development environment and verified basic functionality. The board is ready for more complex sensor integration in future labs.

Key Takeaway: Always verify board selection in Arduino IDE before troubleshooting code!
